# HeavensLive Shop — Global Marketplace

## Overview
A divine marketplace where anyone can buy, sell, and trade globally. Free to join, browse, and list items. 17 languages, 100+ currencies, guest checkout supported.

## How It Works

### For Buyers
1. Browse listings by category, location, or search
2. Filter by price, distance, and type (Mall, Auction, Wanted)
3. Add items to cart — no account required (guest checkout)
4. Message sellers directly for questions
5. Check out with PayPal or Credon Wallet
6. Track purchases and leave ratings

### For Sellers
1. Create a listing in 60 seconds
2. AI Listing Assistant generates title, description, category, and price from plain language
3. Import listings from eBay, Etsy, or Amazon via URL
4. Feature listings to appear at the top of search results
5. Set shipping options — worldwide or local pickup
6. Track analytics and sales in your dashboard
7. Access seller tools: bulk CSV import, promotions, gift cards

## Listing Types
- **Mall** — Fixed price listing with Buy Now
- **Auction** — Bidding with minimum bid and optional buy-it-now
- **Classifieds** — Fixed price, no bidding
- **Wanted / Procurement** — Request items you need

## Key Features
- **Featured Listings**: Appear at top of search (included with Pro/Business plans, or $1/mo)
- **Multi-Currency**: USD, EUR, GBP, BTC, ETH, 100+ currencies with live rates
- **Worldwide Shipping**: Shipping calculator, local pickup support
- **Storefronts**: Every seller gets a customizable store page
- **Escrow Protection**: Secure transactions with dispute resolution
- **Offline Mode**: PowerSync WASM SQLite — browse even without internet
- **Messages**: Real-time chat between buyers and sellers
- **Wishlists**: Save items for later
