📱 Get the App

Install HeavensLive on any device — no app store needed.

🤖 Android

1
Open Chrome on your phone
2
Go to heavnslive.com/shop
3
Tap menu then Add to Home screen
4
Tap Install — opens fullscreen like a native app
Also available on Google Play

🍎 iPhone & iPad

1
Open Safari on your device
2
Go to heavnslive.com/shop
3
Tap the Share button at the bottom
4
Scroll down, tap Add to Home Screen, then Add
Opens fullscreen with the HeavensLive icon. Works offline.

💻 macOS

1
Open Chrome on your Mac
2
Go to heavnslive.com/shop
3
Click the install icon in the address bar
4
Or: File menu → Cast, Save and Share → Create Shortcut
Opens in its own window like a desktop app. Also works in Safari: File → Add to Dock.

🖥️ Windows & Linux

1
Open Chrome or Edge
2
Go to heavnslive.com/shop
3
Click the install icon in the address bar
Desktop app with offline support — no download needed.
